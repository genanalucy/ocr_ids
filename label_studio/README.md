# Label Studio：扁平结构一期

导入 `flat_structure_v1.xml` 作为项目标注界面。每个任务只处理一个候选方块字。该模板同时记录结构标签、规范 IDS 与可用于训练部件定位分支的矩形框。

一期可接受的结构只有：`⿰`（左右）、`⿱`（上下）、`⿲`（左中右）和 `⿳`（上中下）。如果图像不完整、包含多个字、是包围结构或含嵌套结构，选择相应质量标签或 `out_of_scope`，不要勉强填写 IDS。

自动预标注的 `character_hint`、`ids_hint` 和 `components_hint` 仅作提示，标注员必须以图像为准。建议所有 `out_of_scope`、`unreadable` 和人工修改 IDS 的任务进入第二人复核队列。

## 部件框标注规则

1. 先选图像质量；截断、多字、模糊或无法判定的图像不要框部件，直接选择相应质量标签。
2. 再选结构：`⿰`/`⿱` 标两个框，`⿲`/`⿳` 标三个框；包围或嵌套结构选 `out_of_scope`，不标框。
3. `component_1`、`component_2`、`component_3` 的顺序必须与 IDS 的子节点一致：左右结构按左→右，上下结构按上→下，三分结构按视觉顺序。
4. 框应紧贴可见部件的外接矩形，并保留 2–5 像素留白；不框整字，也不把两个部件放进同一个框。
5. 在 `component_1_name`、`component_2_name`、`component_3_name` 中填写对应部件。例如“识”标注为结构 `left_right`，部件 1=`讠`、部件 2=`只`、IDS=`⿰讠只`。
6. 部件存在变形时填写 IDS 使用的部件名称，例如 `讠`、`忄`、`氵`，并在 `note` 中注明不确定性。

建议第一批先标 500–1,000 张真实截图或扫描单字，其中至少一半选自模型错例与 `莫/臭`、`日/目`、`土/士` 等易混淆部件。先由一人标注，再抽取 10% 交叉复核；确认一致后再扩展规模。
