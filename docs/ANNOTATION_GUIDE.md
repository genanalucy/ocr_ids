# 人工标注规范

## 标注对象

人工不逐张标注字体渲染图，只处理：

- 数据源冲突；
- 地区字形与 IDS 不匹配；
- PUA/CDP 或未知部件；
- 多种合理分解；
- 真实未编码字金标准；
- 少量部件框和未知区域。

## 标注步骤

1. 确认图像是一个完整字形，不包含相邻字或切边；
2. 标记地区字形和可追溯来源；
3. 从整体结构开始，以前缀 IDS 构造树；
4. 按当前规范部件表递归拆分；
5. 无法命名的部件用 `？`，同时框选区域；
6. 多种合理分析分别写入主答案和 `alternatives`；
7. 运行语法检查后提交。

## 质量控制

- 金标准由两名标注者独立完成；
- 分歧由第三名专家裁决；
- 不以“能否拼出相似图形”代替字形来源判断；
- 不因已知整字身份反推一个图像中并不存在的部件；
- 每次规范变化必须产生新版本，旧标注不得静默改写。

建议首批金标准为 3,000—5,000 条已编码字形、1,000—2,000 条未见整字组合，以及 500—1,000 条真实未编码字形。
